\chapter*{Executive Summary}
\addcontentsline{toc}{chapter}{Executive Summary}

\noindent
Investors need trusted answers about \emph{their own} holdings---quantities, sectors, allocation, P\&L---without navigating portfolio screens, plus an auditable mood signal, without invented positions or buy/sell advice. This AlgoFabric Industry Sponsored Project (VIT Pune) delivers that on the GenAI Chatbot track, distinct from NL screening and backtesting agents.

\vspace{0.85em}
\noindent
The team built an \textbf{API-first} system that answers only from an ingested \textbf{Neo4j} portfolio graph (user-scoped Cypher GraphRAG), detects mood with \textbf{DistilRoBERTa} (not the LLM), keeps short-term context in \textbf{Redis}, and refuses empty retrieval and advisory language. Orchestration is hand-rolled and packaged with \textbf{Docker Compose}, idempotent JSON ingest, tests, and a minimal test page.

\vspace{0.85em}
\noindent
The MVP ships \texttt{POST /chat}, \texttt{GET /health}, citations, mood gating, dual memory, and sample portfolios (\texttt{demo}, \texttt{u\_alpha}, \texttt{u\_beta}). Live market feeds, host SSO/JWT, vector-DB RAG, full consumer UI, and sibling-agent orchestration were out of scope; Planned items include semantic cache, streaming, LLM observability, and horizontal API scale.
