# AlgoFabric GenAI — Documentation

Documentation for the **AlgoFabric Chatbot** workstream (Portfolio GraphRAG Chatbot).

Sister tracks (NL Query Agent, Backtesting Agent) appear only for scope context.

## Folder structure

```text
docs/
├── README.md
├── meetings/              ← sprint / sync notes
├── development/           ← decisions, timeline, action items
└── engineering-report/    ← SEDD (LaTeX) + image checklist
    ├── latex/             ← compile main.tex (Overleaf)
    ├── IMAGES_NEEDED_FROM_AAYUSH.md
    ├── Images_Needed_for_Documentation.docx
    ├── ROADMAP.md / STATUS.md / README.md
    └── HANDOVER_EMAIL_ATTACHMENT.md
```

## Report (SEDD)

Authoritative report source: **`docs/engineering-report/latex/`**  
Upload that folder to Overleaf → main document `main.tex`.

Image filenames: `engineering-report/latex/images/README.md` or `IMAGES_NEEDED_FROM_AAYUSH.md`.

## Meetings

Store notes under `meetings/` as `Meeting_NN_ShortTitle.md` with date, attendees, decisions, and action items.
